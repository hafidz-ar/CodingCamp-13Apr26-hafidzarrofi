# Product: Expense & Budget Visualizer

A mobile-friendly, standalone single-page web app for tracking daily spending. Users can log transactions (name, amount, category), view a running total balance, browse a scrollable transaction history, and see a pie chart of spending by category.

Key characteristics:
- No backend, no login, no external data — fully self-contained
- All data persisted client-side via `localStorage`
- Zero build tooling required — open `index.html` directly in a browser
- Categories: Food, Transport, Fun
